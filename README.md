# CryptoTrackerPro

RubyGem для анализа криптовалют:
- GUI-режим (Tk)
- консольный режим
- отчеты за 7/30/365 дней
- расчет "машины времени" для инвестиции на год

Репозиторий: `CriptoStats`  
Имя гема: `crypto_tracker_pro`

## Что умеет

- Получает исторические цены (365 дней) через CoinGecko API.
- Считает минимум/максимум/среднюю цену по периодам 7/30/365 дней.
- Показывает тренд в процентах по каждому периоду.
- Считает результат инвестиции "100$ год назад".
- Поддерживает вывод в USD/EUR/RUB (EUR и RUB по фиксированным коэффициентам).

## Установка для разработки

```bash
bundle install
```

## Локальный запуск

### GUI

```bash
bundle exec ruby bin/crypto-tracker-pro
```

### Консоль

```bash
bundle exec ruby bin/crypto-tracker-console
```

## Быстрая проверка перед демонстрацией

```bash
ruby -c lib/crypto_tracker_pro.rb
ruby -c lib/crypto_tracker_pro/*.rb
gem build crypto_tracker_pro.gemspec
```

## Сборка гема

```bash
gem build crypto_tracker_pro.gemspec
```

## Установка локально собранного гема

```bash
gem install ./crypto_tracker_pro-0.1.0.gem
crypto-tracker-console
```

## Примечания

- Для GUI нужен `tk` (в некоторых окружениях его нужно установить отдельно на уровне системы).
- Курсы EUR/RUB внутри приложения заданы статически.
- Для публикации на RubyGems обновите `version.rb` и данные автора в `crypto_tracker_pro.gemspec`.
